# PixiJS TypeScript Starter Template

A PixiJS 2D game and graphics starter template built with TypeScript and Vite.

## Quick Start

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open http://localhost:8080 in your browser.

## Available Scripts

- `npm run dev` or `npm start`: Starts the Vite development server.
- `npm run build`: Runs ESLint, type checks with TypeScript, and bundles for production.
- `npm run lint`: Runs ESLint across the codebase.

## Setup Information

This project was bootstrapped using PixiJS CLI:

```bash
npm create pixi.js@latest .
```

- Template: Bundler Templates
- Variant: Vite
